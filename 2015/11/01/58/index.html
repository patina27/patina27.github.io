BZOJ4300: 绝世好题

发表于 2015-11-01   |  

Description

给定一个长度为n的数列ai，求ai的子序列bi的最长长度，满足bi&bi-1!=0(2<=i<=len)。

Input

输入文件共2行。
第一行包括一个整数n。
第二行包括n个整数，第i个整数表示ai。

Output

输出文件共一行。
包括一个整数，表示子序列bi的最长长度。

Sample Input

3

1 2 3

Sample Output

2

HINT

对于100%的数据，1<=n<=100000，ai<=10^9。
